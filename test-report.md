## 3. 极端情况测试

- **词汇量不足 20**: 章节列表应只显示 1 章，范围为 1-N。 (已验证)
- **非字母拼音**: 某些词典可能包含特殊字符。 (已处理：排序和过滤 logic 已考虑)
- **快速切换章节**: 连点不同章节。 (已处理：`restartPracticeMode` 会清理旧状态)
- **词典加载失败**: 网络问题或文件损坏。 (已处理：有错误提示 `showErrorMessage`)

## 4. 结论
功能实现完整，UI 现代美观，符合用户对于“章节化练习”的需求。代码逻辑稳健，具备良好的错误处理。
