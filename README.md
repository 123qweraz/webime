# WebIME Pro 🚀

一款基于 Web 技术构建的极简、高效、可定制化云输入法。

## 🌟 核心亮点 (Key Highlights)

* **⚡ 极致速度 (Ultra-fast)**：原生 JavaScript 编写，无冗余库，毫秒级响应，即使在老旧浏览器上也能流畅运行。
* **📖 英语辅助编码 (English-Aided Coding)**：**国内首创**的功能！汉字候选词下方实时显示对应的英文翻译。不仅能辅助打字，还能在打字的过程中潜移默化地学习英语词汇。
* **🎯 Tab 键逻辑过滤 (Tab-Mode Filtering)**：输入拼音后按下 `Tab`，可根据英语含义的开头字母进行二次精确过滤。输入 `hao` + `Tab` + `g`，直接秒定“好 (good)”，极大减少翻页次数。
* **🛠️ 三级字典加载 (Three-Level Dictionary)**：支持多层级词库切换，从基础字库到深度专业词库，性能与容量兼得。
* **⌨️ 全键盘快捷操作 (Keyboard First)**：
* `Ctrl + E`：在**打字模式**与**自由编辑模式**间一键切换，边打边改，无需鼠标。
* `Ctrl + C`：一键**归档并复制**，快速清理屏幕并将内容存入历史记录。
* `-` / `=`：经典的翻页逻辑，符合传统输入法习惯。


* **☁️ 云端归档 (Cloud-Ready Archive)**：内置历史记录面板，自动保存最近的输入片段，防止灵感丢失。
* **✨ 极简 UI (Minimalist Design)**：无广告、无弹窗、无后台常驻。只有你和文字，提供纯粹的写作体验。

## 🚀 快速开始 (Quick Start)

1. **克隆项目**：
```bash
git clone https://github.com/123qweraz/webime.git

```


2. **运行**：
直接用浏览器打开 `index.html` 即可开始打字。
3. **自定义词库**：
将你的 `.json` 格式词库放入 `dicts/` 目录下即可完成扩充。

## 🎹 常用快捷键 (Shortcuts)

| 快捷键 | 功能说明 |
| --- | --- |
| **Tab** | 进入/退出英语辅助过滤模式 |
| **Ctrl + E** | 切换【锁定打字】与【自由编辑】模式 |
| **Ctrl + C** | 归档当前内容到左侧历史记录并复制到剪贴板 |
| **Esc** | 在编辑模式下快速返回打字状态 |
| **数字 1-0** | 选择对应的候选词（Buffer 为空时可选常用字） |
