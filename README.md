# WebIME 🚀

一款基于 Web 技术构建的极简、高效、深度可定制的跨平台输入法框架。

## 🌟 核心功能 (Features)

- **⚡ 极速响应**：基于 Trie 树的高效检索，支持多词库并发匹配，毫秒级候选词呈现。
- **📝 长句模式 (Long Sentence Mode)**：支持长拼音串输入，智能切分并提供分段候选词修正，模拟 Vim 风格的快速编辑体验。
- **🏋️ 练习系统 (Practice Suite)**：内置科学的打字练习流程，支持章节划分、进度持久化及实时纠错反馈。
- **📚 动态词典管理**：支持内置词库（中/日/英）与用户自定义 JSON 词库的实时开关与混合输入。
- **⌨️ 英语辅助输入 (English-Assisted Input)**：独创的 `Tab` 与 `Shift` 联动机制，支持通过英文语义快速过滤候选词，并一键输入对应的英文释义。
- **🔍 汉字浏览器 (Explorer)**：可视化汉字结构，支持汉字的组合、拆解与排版实验。
- **📋 归档工作流**：一键“归档并复制” (Ctrl+C)，自动保存历史记录，防止灵感丢失。
- **🌓 现代 UI/UX**：支持深色/浅色模式切换，响应式布局，且具备可调节的输入/输出区域。

## 🎹 快捷键手册 (Shortcuts)

| 快捷键 | 功能说明 |
| :--- | :--- |
| **Tab** | **中英切换**：在输入过程中快速切换输出内容（中文词组 ↔ 英文释义） |
| **Shift** | **英语辅助模式**：进入/退出基于英文释义的过滤模式 |
| **Double Shift** | 在过滤模式下，快速选中并输入首个候选词的英文释义 |
| **Ctrl + I** | 进入 **长句模式** ( correction mode ) |
| **Ctrl + E** | 切换 **锁定模式** (输入) 与 **编辑模式** (自由修改) |
| **Ctrl + C** | 归档当前内容并复制到剪贴板 |
| **Esc** | 在任意模式下快速返回或同步缓冲区 |
| **F2** | 练习模式下切换拼音提示显隐 |
| **- / =** | 候选词翻页 |

## 💡 特色功能详解：英语辅助输入

WebIME 提供了强大的“以英搜中”与“中英混输”能力：

1. **Shift 辅助过滤**：输入拼音后按下 `Shift`，即可输入英文关键词过滤当前的中文候选词。例如输入 `pingguo` 后按 `Shift` 再输入 `apple`，可精确锁定“苹果”。
2. **Tab 灵动切换**：在输入过程中按下 `Tab`，候选词面板将实时在“中文原词”与“英文注释”间切换。配合数字键即可直接输入对应的英文单词。
3. **高效双击**：在 `Shift` 模式下双击 `Shift` 键，可直接上屏首位候选词的英文释义，极大地提升了中英翻译与写作的连贯性。

## 🛠️ 技术架构 (Architecture)

- **核心引擎**：原生 JavaScript (ES6+)，无第三方框架依赖。
- **数据结构**：采用前缀树 (Trie) 实现多级词典的快速检索与权重排序。
- **持久化**：使用 `localStorage` 存储用户偏好、词典配置及练习进度。
- **模块化设计**：
  - `ime.js`: 核心输入状态机与候选词逻辑。
  - `trie.js`: 高性能检索树实现。
  - `practice.js`: 练习模式状态管理。
  - `dict-manager.js`: 词库加载与切换调度。

## 🚀 快速开始

1. **直接运行**：克隆仓库后，双击 `index.html` 即可在浏览器中使用。
2. **导入词库**：在“词典设置”中上传符合 `{ "pinyin": ["char1", "char2"] }` 格式的 JSON 文件。
