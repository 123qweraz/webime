# webime - 基于网页的输入法编辑器
一款轻量、原生网页版输入法编辑器（IME），支持英文输入与标点符号输入，专为网页应用无缝集成设计。

## 概述
webime 是一款基于前端技术构建的极简网页输入法解决方案。它提供基础的英文输入和标点符号输入功能，让开发者无需依赖系统级输入法工具，即可在网页中嵌入一个简单、独立的输入法。

该仓库包含输入法运行所需的核心文件，无外部依赖项——部署和自定义都十分便捷。

## 文件结构
仓库包含两个关键文件：
| 文件名       | 描述                                                                 |
|--------------|----------------------------------------------------------------------|
| `index.html` | 主入口文件，包含输入法界面和核心逻辑。直接在浏览器中打开该文件即可使用输入法。 |
| `dict.json`  | 字典文件，支持英文输入和标点符号映射，为输入法的输入识别功能提供支持。       |

## 核心特性
- **英文输入支持**：流畅输入英文字母、数字及基础符号。
- **标点符号支持**：涵盖常用标点符号，满足完整文本输入需求。
- **原生网页属性**：完全在浏览器中运行，无需服务器依赖或系统安装操作。
- **轻量高效**：文件体积小，加载速度快，资源占用低。

## 使用方法
1. 将本仓库克隆到本地：
   ```bash
   git clone https://github.com/123qweraz/webime.git
   ```
2. 进入仓库目录：
   ```bash
   cd webime
   ```
3. 在任意现代浏览器（Chrome、Firefox、Safari、Edge 等）中打开 `index.html` 文件。
4. 在输入法输入区域直接打字，即可识别并显示英文字符和标点符号。

## 自定义扩展
- **扩展字典**：修改 `dict.json` 文件，可添加更多英文单词、短语或自定义标点映射。文件采用简洁的 JSON 结构，便于编辑。
- **界面调整**：编辑 `index.html` 中的 HTML/CSS 代码，可自定义输入法的外观（如尺寸、颜色、布局等），适配不同网页场景。
- **功能扩展**：在 `index.html` 的逻辑代码中添加新特性，例如输入联想、快捷键支持等（需具备基础前端开发知识）。

## 注意事项
- 本输入法目前仅支持英文和标点输入，暂不提供中文等其他语言输入功能。
- 建议在现代浏览器中使用，以确保功能正常运行。
- 如需用于生产环境，可根据实际需求优化代码（如压缩文件、添加兼容性处理等）。
